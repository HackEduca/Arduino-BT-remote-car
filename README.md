# Hardware

- Arduino Uno
- L293D motor controller
- HC-06 Bluetooth slave serial
- Battery: 5v for Arduino and 9v for L293D
- Two 55rpm/2.3kgf.cm Motors with wheel made from [icci Inc.](http://www.icci.com.tw)
- Some wires and tools
